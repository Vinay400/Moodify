# Moodify — Emotion-Based Music Recommendation System

> A real-time facial emotion recognition system that recommends music based on detected emotions, using a CNN trained on FER-2013 and the MUSE v3 music dataset with valence/arousal mapping.

---

## Abstract

This project presents **Moodify**, an intelligent music recommendation system that bridges affective computing and music information retrieval. The system captures a user's facial expression via webcam, classifies it into one of seven emotion categories using a Convolutional Neural Network (CNN), and recommends songs whose emotional profile matches the user's current mood. Recommendations are generated by mapping detected emotions onto Russell's circumplex model of affect (valence × arousal) and filtering the MUSE v3 dataset accordingly.

## Methodology

### 1. Face Detection
The system uses OpenCV's **Haar Cascade Classifier** (`haarcascade_frontalface_default.xml`) to detect faces in each video frame. Histogram equalisation is applied to improve robustness under varying lighting conditions. When multiple faces are present, the largest bounding box is selected.

### 2. Emotion Recognition (CNN)
Detected faces are cropped and resized to **48 × 48 grayscale** images, then fed into a Convolutional Neural Network with the following architecture:

| Layer | Type | Output Shape |
|-------|------|-------------|
| 1 | Conv2D (32, 3×3) + ReLU | 46×46×32 |
| 2 | Conv2D (64, 3×3) + ReLU | 44×44×64 |
| 3 | MaxPooling2D (2×2) | 22×22×64 |
| 4 | Conv2D (128, 3×3) + ReLU | 20×20×128 |
| 5 | MaxPooling2D (2×2) | 10×10×128 |
| 6 | Conv2D (128, 3×3) + ReLU | 8×8×128 |
| 7 | MaxPooling2D (2×2) | 4×4×128 |
| 8 | Dropout (0.25) | 4×4×128 |
| 9 | Flatten | 2048 |
| 10 | Dense (1024) + ReLU | 1024 |
| 11 | Dropout (0.5) | 1024 |
| 12 | Dense (7) + Softmax | 7 |

The model is trained on the **FER-2013** dataset (35,887 images, 7 classes) and outputs a probability distribution across: *Angry, Disgusted, Fearful, Happy, Neutral, Sad, Surprised*.

### 3. Emotion-to-Music Mapping (Russell's Circumplex Model)
Each emotion is mapped to a valence (pleasantness) × arousal (energy) range based on Russell's circumplex model of affect:

| Emotion | Valence Range | Arousal Range |
|---------|:------------:|:------------:|
| Happy | 5.8 – 9.0 | 4.3 – 9.0 |
| Sad | 0.0 – 4.2 | 0.0 – 3.8 |
| Angry | 0.0 – 4.5 | 5.0 – 9.0 |
| Fearful | 0.0 – 4.5 | 3.0 – 5.5 |
| Neutral | 4.0 – 6.8 | 3.2 – 5.2 |
| Surprised | 4.5 – 8.0 | 5.0 – 9.0 |
| Disgusted | 0.0 – 3.8 | 3.5 – 6.0 |

### 4. Recommendation Algorithm
1. Detected emotions are ranked by frequency of occurrence during the scan
2. Song candidates are filtered by valence/arousal range and optional genre constraints
3. Weighted sampling assigns more tracks to the primary emotion
4. Results are deduplicated and limited to the user-specified count

## Dataset

**MUSE v3** (MUsic and Sentiment/Emotion) — a crowd-sourced music emotion dataset:

| Property | Value |
|----------|-------|
| Total tracks | ~90,000 |
| Unique genres | 811 |
| Features per track | name, artist, genre, valence, arousal, dominance |
| Source | Last.fm + Spotify metadata |

## Features

- **Real-time emotion scanning** via webcam with live preview
- **CNN confidence visualisation** — horizontal bar chart of probabilities
- **Valence/arousal-based recommendations** using Russell's circumplex model
- **Genre filtering** — filter songs by 40+ genres
- **Adjustable recommendation count** (5–50 songs)
- **Manual mood override** — select emotions without camera
- **Favorites system** — save and export liked songs
- **Mood Journal** — log and review scan history
- **Analytics Dashboard** — mood distribution pie chart, timeline scatter plot, summary statistics
- **About & Methodology page** — full academic documentation within the app
- **Spotify & YouTube links** for every recommendation
- **CSV export** for recommendations and favorites

## Installation & Setup

### Prerequisites
- Python 3.10 or higher
- Webcam (for live emotion scanning)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hiralsathwara/Emotion-based-music-recommendation-system.git
   cd Emotion-based-music-recommendation-system
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   streamlit run app.py
   ```

4. Open `http://localhost:8501` in your browser.
